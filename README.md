# task-todo

A simple UI to manage tasks

## Prerequisites

At least Python 3.7

## Project Setup & Commands

### Install venv

```shell
python3 -m venv .venv
```

### Install required library

```shell
source .venv/bin/activate

pip install -r requirements.txt

# For newer Mac
brew install python-tk
```

### Setup required files

```shell
sh ./setup.sh
```

### Run the program

```shell
python3 ui.py
```

### Add new task

```shell
sh ./add_task.sh
```
